# FizzBuzz問題

## 課題内容
自然数が与えられます。以下のルールに従って、1から与えられた数字`n`まで順に表示する関数を作成してください。

- 3の倍数の場合は`Fizz`
- 5の倍数の場合は`Buzz`
- 15の倍数の場合は`FizzBuzz`
- 上記以外の場合はその数字をそのまま表示

## 入力例

```js
15
```

## 出力例

```js
1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz
```

## 提出方法
- `fizzbuzz.js`を編集し、提供されたテストをすべて通過させてください。

### テスト実行方法

```bash
npm test -- fizzbuzz/fizzbuzz.test.js
```
